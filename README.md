### 🎯 About Me

I'm a B.Tech student in Artificial Intelligence and Data Science, currently working as an **AI Engineer Intern**, building ML models for drug discovery. My research sits at the intersection of **Explainable AI, NLP/LLMs, RAG, and multi-agent systems**, and I've published across IEEE and Springer venues while still an undergrad.

I like problems where the model has to be both *accurate* and *interpretable* — and I care about building systems that people can actually trust and understand, not just black boxes that perform well on a leaderboard.

### 🚀 Featured Projects

- **[CCDS](https://github.com/kundanbedmutha/CCDS-AI)** — Causal-Counterfactual Decision Support framework for Explainable AI. Generates SCM-grounded counterfactual recourses across 3 UCI benchmark datasets, evaluated against 5 baselines with a novel Causal Consistency Fidelity (CCF) metric. Published at IC3SIS 2026 (IEEE).

- **[ApologySense](https://github.com/kundanbedmutha/ApologySense)** — Apology detection using a hybrid BERT model, published at ICETICS 2026 (IEEE).

- **HybridMem** — Agent memory architecture library combining hybrid vector + knowledge-graph memory, RL-driven memory management, multi-agent semantic consistency, and GDPR-aligned privacy controls. *(repo link coming soon)*

- **Drug Discovery ML** *(AI Engineer Intern)* — Building end-to-end ML pipelines for molecular property prediction and drug-target interaction modelling at scale.
